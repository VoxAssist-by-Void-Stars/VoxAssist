# Marcus — Past Projects

## PulseNode
A sensor node that samples telemetry on a real-time schedule and publishes over MQTT. C on STM32 +
FreeRTOS. I was obsessed with timing determinism — jitter was the enemy.

## FerroDrive
A brushless-motor controller firmware, rewritten in **Rust** for memory safety. Learned to love
Rust's guarantees in a domain where a crash is a physical event, not a stack trace.

## TinyLog
A tiny, allocation-free logging library for embedded targets. Ring-buffer based, zero dynamic
memory. My "constraints breed good design" project.
