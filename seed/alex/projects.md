# Alex — Past Projects

## TrailMate
A hiking-route planner. Next.js + Supabase + Mapbox. Users save routes and share GPX files. Taught
me geospatial queries in Postgres (PostGIS) and how much UX polish maps demand.

## Inkwell
A minimal markdown blog engine (Next.js, MDX). Static-first, near-zero client JS on content pages.
My "how simple can it be" project — I kept deleting code until it broke, then added one line back.

## Standup Bot
A Slack bot (Node + Bolt) that collects async standups and posts a daily digest. Lesson: keep
integrations dumb and stateless; store as little as you can get away with.
