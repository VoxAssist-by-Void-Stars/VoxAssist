# Priya Nair — Profile

## About
ML and data engineer, 5 years. I live between research and production — taking a messy notebook idea
and turning it into a reproducible pipeline. **Reproducibility is non-negotiable**: fixed seeds,
pinned configs, versioned data.

## How I like to work
- **Notebook-first exploration**, then refactor into clean scripts/pipelines once the approach is proven.
- I write **RFCs / design docs** before big changes, and I expect a clear problem statement —
  vague tickets frustrate me.
- Async code review, thorough. I read carefully and comment a lot; it's care, not criticism.
- Slightly nocturnal — my best focus is 9pm–1am.

## Conventions
- Type hints + thorough docstrings on every public function.
- Config over hardcoding (Hydra/YAML). Everything reproducible from a single command.
- Small, tested data transforms over one giant script.

## Hobbies
Heavy euro board games (Terraforming Mars is a favorite), sourdough baking, and weekend birdwatching.
