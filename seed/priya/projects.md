# Priya — Past Projects

## ChurnScope
A customer-churn prediction pipeline. Airflow + scikit-learn + BigQuery. The focus was a clean,
reproducible training→eval loop and honest metrics — I spent most of the time hunting data leakage.

## LeafID
A CNN (PyTorch) that classifies plant-leaf diseases from photos. Hard lessons in class imbalance and
data augmentation; the model was easy, the dataset was the real project.

## MetricMuse
The backend for an experiment-tracking dashboard — a FastAPI service over Postgres storing runs and
metrics. I built the API; someone else did the UI (gladly).
